Code and results for ECON 8210 Problem Set 1

Folder "code" include all the codes:
hw1_q2: numerical integral question Q2
hw1_q3: optimization question Q3
hw1_q4: social planner allocation question Q4
hw1_q5: equilibrium price computation question Q5
hw1_q6: main file for the VFI question Q6
q6_ge_func: contains functions needed to compute the general equilibrium when using fixed grids
q6_ge_func_stochastic: contains functions needed to compute the general equilibrium for Q6 when using stochastic grids

Folder "results" include all the codes:
HW1_Report: main report
The remaining figures or tables are named by its corresponding question number.
